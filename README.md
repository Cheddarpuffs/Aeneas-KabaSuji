# Aeneas-KabaSuji
Aeneas KabaSuji


## Building
./gradlew build

## Running
./gradlew run

## With Eclipse

If you want to use eclipse and gradle:

 - Install eclipse gradle plugins
 - Run as Gradle STS

If you just want to use it as a java project

 - file > import > existing project
 - Run as Java Application
